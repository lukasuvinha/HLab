# Home Server Infra Base

## Objetivo
Detalhar as etapas de criação de um servidor Linux com Docker e backup automático

## Tecnologias usadas
- Ubuntu Server
- Docker
- Nginx
- rsync

## Etapas realizadas
1. Instalação do Ubuntu
2. Configuração SSH
3. Instalação Docker
4. Criação servidor web
5. Backup automático
